# Example-API-CRUD
> Example-API-CRUD é uma API, desenvolvida em JavaScript com nodeJS e express, apenas para manipular dados estáticos para entender as operações CRUD (Create, Read, Update and Delete).


## Instalação dos componentes necessários

Para testar as rotas:

- [Insomnia](https://insomnia.rest/download/#windows/)


## Requisições suportadas pela API

GET /projects

```sh
Retorna os dados de projetos cadastrados.
```

POST /projects

```sh
Cria novos registros de projetos.
```

Put /projects/:id

```sh
Atualiza os dados do projeto com o id passado na requisição.
```

Delete /projects/:id

```sh
Deleta o projeto com o id passado na requisição.
```


> Ps. Lembrando que esse projeto tem como intuíto mostrar como funcionam as rotas em API's, apenas.
